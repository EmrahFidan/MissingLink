# MissingLink Frontend

Next.js + TypeScript + Tailwind CSS tabanlı yönetici paneli.

## Kurulum

```bash
npm install
```

## Geliştirme

```bash
npm run dev
```

Tarayıcıda açın: [http://localhost:3000](http://localhost:3000)

## Ortam Değişkenleri

`.env.local` dosyası oluşturun:

```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

## Özellikler

- CSV dosya yükleme
- Veri analizi ve istatistikler
- Responsive tasarım
- TypeScript tip güvenliği
- Tailwind CSS ile modern UI
